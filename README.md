# toDoList
Goal: Simple to do list for browser, where we can add new notes, delete or edit existing ones with the feature to automaticly all locally

Current state: 
1)We have an input zone
2)By entering an input we get the note displayed
3)Using X button notes can be deleted
4)Pressing Edit button opens a new input section with current notes and a save button to save changes
5)Pressing Save button saves changed text from editing  
6)All these action will save locally to keep always track of your notes
